# mini-projet-analyse-donnees
Mini-Projet – Analyse Automatique de Données (Architecture)

## Description du projet

Ce mini-projet a pour objectif de développer un script Python permettant de réaliser une analyse automatique de données de recherche dans le domaine de l’architecture et des matériaux de construction.

Le dataset utilisé contient des informations sur différents types de bâtiments ainsi que les quantités de matériaux de construction utilisés (béton, acier, bois, verre, brique, ciment).

Le script développé sous Google Colab permet de :

- Lire automatiquement un fichier CSV hébergé en ligne
- Nettoyer les données (gestion des valeurs manquantes et conversion des types numériques)
- Organiser les données à l’aide de DataFrames (Pandas)
- Effectuer des analyses statistiques (moyennes, corrélations, statistiques descriptives)
- Générer des visualisations graphiques (diagrammes, heatmap, boxplots)
- Exporter les résultats vers un fichier Excel

L’objectif principal est d’automatiser le processus de traitement et d’analyse des données afin de produire des résultats fiables et exploitables dans un contexte de recherche.
